Girls Go Tech

Welcome to the Girls Go Tech Coding Workshop! You will learn how to read and write code, build your own website, and design your own game!

Let's get started!

Workshop Materials

These are the websites and applications you will need for this workshop. Click the images to go to each site!

CodePen

This is a website that allows you to test out code in real-time. We will be using this a lot!

Sublime Text

This is a free text-editor app you will use to write the code for your website and game.

HTML5Up

This is a site that provides free website templates for you to use for your own website.

GitHub

This is what we'll be using to publish your website for free! You will need to sign up for an account.

GitHub Desktop App

Download the GitHub app here after signing up for your GitHub account. You will use this to manage your site.

WordPress

This is the website you will be using to write your blogs at the end of each day.
