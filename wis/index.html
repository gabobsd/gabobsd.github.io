Downloads

These are the apps and websites you'll need for this week.

CodePen

A "code playground" to let you play around with code and test it out.

Sublime Text 3

A text editor app you'll use to write your code for your website and game.

A site to download free templates which you'll use to build your first website!

GitHub

WordPress
