Downloads

These are the apps and websites you'll need for this week.

CodePen

Codepen is a "code playground" - it lets you play around with code and test it out. You'll be using this a lot.

Sublime Text 3

A text editor app that you'll be using to write your code. You'll be using this app to build your website and game.

HTML5Up

This is a site that lets you download free templates for you to use to build your own website!

GitHub

This what we'll be using to publish your site! You'll need to register for an account on GitHub AND download the desktop app HERE.

GitHub Desktop

After you sign up for a GitHub account, you'll need to download the desktop app here.

WordPress

This is the site you'll be using to write your blogs at the end of each day.
